Tentang Web Ini

Web ini mulai dibuat pada tanggal 27 Januari 2017, berawal dari hobi dengan kue, dan tema UAS yang berhubungan dengan kue mendorong saya untuk membuat web ini.
Saya membuat website ini diidekan oleh dosen saya, sebagai tugas UAS dengan disediakan 40 tema yang hanya dapat dipilih oleh 1 mahasiswa saja
dan bilamana web ini di upload di github akan mendapatkan nilai 100.
Dengan meminjam gambar dan konten dari website lain, akhirnya terciptalah web fiksi ini.

Proses pengerjaan website ini memakan waktu sekitar seminggu, dimulai dari pencarian gambar, lalu menentukan nama usaha,
dilanjut pembuatan logo, pembuatan tampilan Beranda, pengeditan gambar dengan cara croping.
Kembali ke pembuatan kontak, insert peta lokasi. Pembuatan Galeri dan Profil, dan terakhir Footer.
Lalu muncul permasalahan dimana foto Galeri yang tidak mau bergeser kesamping tetapi malah turun ke bawah.

Hingga akhirnya selesai juga web ini pada 6 Februari 2017
Akhir kata masih banyak kekurangan pada web ini yang kedepannya akan jadi lebih baik lagi.
Terima kasih kepada dosen saya Dimas Widra Yato atas ilmunya selama ini, asik belajar membuat web dan ilmu yang sangat bermanfaat untuk bekal bekerja setelah lulus.
